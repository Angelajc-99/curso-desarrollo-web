El operador de asignación

En JavaScript, el signo de igual = no equivale a "resultado de" como en álgebra. Si no que es un operador de asignación.

x = x + 5

Esta expresión, matemáticamente, no tiene sentido. Pues nunca va a haber un número equivalga a una suma de sí mismo con 5.

Sin embargo, en código, no solo tiene mucho sentido si no que es una práctica muy común. Esta expresion indica que al actual valor de x le estamos asignando el valor de la suma de x + 5.

Básicamente hemos aumentado el valor de x en 5.

Nota: en JavaScript el operador "igual a" se escribe ==.

Tipos de datos

Las variables pueden tener distintos tipos de datos. Pueden ser número como 100 o texto como "Juan Eduardo".

En programación, a los valores se les llama cadenas de caracteres a strings.

Los strings se escriben dentro de las comillas dobles o simples.

Si escribes un número dentro de unas comillas, será tratado como un string de texto. Ref al js línea 15
