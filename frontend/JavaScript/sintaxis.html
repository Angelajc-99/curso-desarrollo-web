Variables

En progrmación las variables se usan para almacenar valores.

JavaScript se sirve de las Keywords var, let, const para declarar variables.

El signo = se usa para asignar valores a las variables.

JavaScript usa operadores aritméticos (+ - *) para computar valores.

Expresiones

Comentarios

Los comentarios son líneas de código que son ignoradas. No se ejecutan.

Tienen principalmente dos usos.El primero es explicar el código y el segundo es para desactivar cierta parte de nuestro de código sin necesidad de borrarla con /*...*/

Identificadores/Nombres

Los identificadores se usan para nombrar variables y funciones.

Hay una serie de reglas para nombrar comunes en la mayoria de lenguajes de programación.

Un identificador debe empezar por:

  • Una letra (A-Z o a-z)
  • Un simbolo de dolar $
  • O una barra baja_

Los caracteres siguientes pueden ser números, letra, _, $.

No se permite empezar con números!
