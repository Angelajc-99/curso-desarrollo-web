"Layout" en CSS

El layout es la disposición o la forma en la que los elemntos se muestran en panatalla según su tipo.

Display

Es la propiedad más importante de CSS para controlar el alineamiento.

La propiedad display especifica la forma en la que se muestra un elemento.

En HTML cada elemento posee un valor por defecto dependiento del tipo de elemento que sea. La mayoría de los elementos poseen una propiedad block o inline por defecto.

Elementos de bloque

Los elementos del bloque siempre empiezan en una nueva línea y toman toman todo el ancho disponible.

Este elemento div es de tipo bloque.

Algunos elementos de bloque:

  • div
  • h1 - h6
  • p
  • form
  • header
  • section
  • footer

Elementos en línea (Inline)

Estos elementos no empiezan en una nueva línea y ocupan sólo el ancho necesario.

Esto es un espan dentro un párrafo.

Algunos de estos elementos:

  • span
  • a
  • img

Diferencia entre display: none y visibility:hidden

Al ocultar un elemento con display: none la página se mostrará como si el elemnto no estuviera ahí.

Al ocultar un elemento con visibility: hidden el elemento no se mostrará pero el espacio que ocupa quedará en blanco.

La propiedad position

Esta propiedad define el método de posicionamiento de un elemento.

Hay 5 tipos de posición.

  • static
  • relative
  • fixed
  • absolute
  • sticky
    •

Los elementos se posicionan con las propiedades top, left, right, bottom. Sin embargo estas propiedades no funcionarán sin especificar primero la position. Además dependiendo del tipo de position las propiedades funcionarán de manera distinta.

position: static

Todos los elementos tienen este valor por defecto.

Las propiedades top, bottom, left y right no afectan a los elementos con posición estática.

position: relative

Un elemento con este valor se posicionará d emanera relativa a su posición normal. Es decir, el centro del eje será sobre sí mismo.

Este div tiene position: relative

position: fixed

Un elemento con este valor se posicionará relativamente respecto a la pantalla. Lo que quiere decir que tendrá siempre la misma posición incluso si scrolleamos la web.

position: absolute

Un elemento con este valor se posicionará respecto a su contenedor posicionado(no estático).

Si un elemento absoluto no tiene ancestro posicionados, usará el body y actuará como un position: fixed.

Este elemento div tiene una position: relative
Este elemento div tiene una position: absolute

position: sticky

Un elemento position: sticky alternará su comportamiento entre relative y fixed dependiendo de la posición de scroll de la página.

El posicionamiento vertical con z-index

El z-index determina el orden en el que se amontonan los elementos.

Cuando los elementos se posicionan, puede que se monten sobre otros.

Esta propiedad determina en que orden se amontonan los elementos. Es decir que elementos van por encima o debajo de los otros.
