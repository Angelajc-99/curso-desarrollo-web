Atributos de elementos HTML

Los atributos proveen infromación adicional de elementos HTML.

  • Todos los elementos HTML pueden tener atributos.
  • Los atributos proveen info adicional d elos elemntos.
  • Los atributos se especifican en la etiqueta deapertura
  • Los atributos normalmente tienen estructura de nombre="valor"

Ejemplo. Un link se define con la etiqueta <a>. El tributo "href" especifica la URL a la página a a que irá el link.
