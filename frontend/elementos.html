Links o hipervínculos

Los link se encuentran en todo tipo de paginas. permiten al usuario navegar por los documentos de la web.

Cuando clicas en un link te mueve a otro documento o a una parte concreta del mismo.

cuando se pasa el cursor sobre un link, este se torna en una mano.

Un link se define mediante un elemento <a>

La sintaxis de un link, por norma general, es: <a href="url>Text del link</a>"

El atributo mas importante de un link es el "hrf" ya que este indica el punto al que nos llevaráel link.
