Definir estilos del HTML con CSS

CSS(Cascade style sheets) se usa para controlar el formato visual de una pagina.

Con CSS se puede controlar el color, fuente, tamaño del texto, espaciado, alineamiento, imagenes de fondo, como se ve una pagina dependiendo del del tamaño de pantalla del dispositivo que estemos usando, etc.

Formas de usar el CSS

  • Inline - usando el atributo style dentro del elemento HTML.
  • Internal - Usando la etiqueta <style> dentrode la seccion del <head>
  • EXternal - Usando un archivo CSS externo.

Internal CSS

Se usa para aplicar estilos a un documento en concreto y se define dentro de la etiqueta >style> dentro de la sección head del documento. Ref en la línea 9 del codigo del documento.

External CSS

Es un archivo con la extensión. CSS que contiene exclusivamente código en CSS. Se puede aplicar al documento mediante el elemnto <link> Ref en la línea 8.

Las dos principales ventajas de externalizar el CSS son:

  • Los estilos se pueden aplicar a tantos documentos como queramos.
  • La longitud del código disminuye sensiblemente
